# Offroad Odyssey 🚙

**Offroad Odyssey** is a **3D car simulator game** built using the **Unity Engine** and **C# scripting**.  
This repository contains the **compiled Windows build** of the game.

---

## 🎮 About the Game

Offroad Odyssey is a simple **3D driving simulator** where players can control a vehicle and explore terrain using realistic vehicle physics.

---

## 📂 Repository Structure

```
Offroad-Odyssey
│
├── Offroad Odyssey.exe
├── MonoBleedingEdge/
│
└── Offroad Odyssey_Data/
    ├── Managed/
    └── Resources/
```

### Folder Explanation

- **Offroad Odyssey.exe** → Main game executable file  
- **MonoBleedingEdge** → Runtime files required by Unity to run the game  
- **Offroad Odyssey_Data** → Game assets, scripts, and resources compiled by Unity  

⚠️ These folders must stay together for the game to run correctly.

---

## ▶️ How to Run the Game

1. Download or clone the repository

```
git clone https://github.com/suryaprakash-s-2005/Offroad-Odyssey.git
```

2. Open the downloaded folder.

3. Run the game by double-clicking:

```
Offroad Odyssey.exe
```

| Key | Action            |

| S   | Brake / Reverse   |
| A   | Turn Left         |
| D   | Turn Right        |
